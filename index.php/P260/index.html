CountDownr
5.0 

CountDownr provides a collection of countdown counters. Each counter is customized to its own time and date when it goes off. There are two types of counters:

Counters for a specific date and time. These counters can be repeated daily, weekly, monthly or yearly which advance automatically when the alert goes off. They are perfect for reminders on single events, lunchbreaks, weekends, paying your monthly bills or birthdays and holidays. Example: Midnight

Counters which read the upcoming events from public Google Calendars. Any public calendar can be the source for a counter. It advances automatically to the next event after each alert. Example: Google Developer Calendar
